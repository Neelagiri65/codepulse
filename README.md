# CodePulse

Public leaderboard and paste-audit for `CLAUDE.md` files across GitHub. Scores configs against documented Claude Code defaults to surface redundant, outdated, or cargo-culted instructions.

- Scope for v0.1: `docs/prd-v0.1-leaderboard-and-audit.md`
- North-star vision: `CODEPULSE_V2_FULL_SPEC.md`
- Work breakdown: `docs/issues-v0.1-leaderboard-and-audit.md`

## Run locally

```sh
pnpm install
pnpm dev       # dev server
pnpm build     # static site → dist/
pnpm test      # unit tests
```
